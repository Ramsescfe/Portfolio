Monstro do Lixo

O Monstro do Lixo é um projeto interativo que utiliza Arduino para educar crianças sobre o descarte correto de resíduos. Ele inclui sensores que detectam o lixo, sons interativos e uma estrutura visual divertida para chamar atenção sem ser invasiva.

Tecnologias usadas: Arduino IDE, sensores, programação embarcada, C,
